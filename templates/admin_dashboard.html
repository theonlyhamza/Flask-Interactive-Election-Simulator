Election Management System

Administrative Control Panel

System Overview

This dashboard allows authorized administrators to manage voters, political parties, constituencies, monitor live election results, and officially conclude the voting process.

Register Voter

Add and manage eligible voters by CNIC, area, and voting status.

Political Parties

Register and manage political parties participating in the election.

Election Areas

Create and manage constituencies / electoral areas.

Live Results

View real-time vote counts and party-wise statistics.

Download Voters Data

Export registered voters data for audit and verification.

Download Votes Data

Export cast votes data for result verification.

End Voting & Declare Results

Permanently close voting and declare final winners.



Important Notice

Ending the voting process is irreversible. Please ensure all votes have been cast before proceeding.
